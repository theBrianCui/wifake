Getting Started

This process asks for your personal information so that we can determine whether you already have a UT EID. This is important because creating a new UT EID when you already have one can cause problems or delays in accessing university services.

If we determine that you already have a UT EID, we will connect you to your existing UT EID. Otherwise, we will use the information to guide you through the process of creating a new UT EID.
